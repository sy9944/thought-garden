# thought-garden
A personal journal app for cultivating thoughts over time.

時間をかけて思考を育てていくための日記アプリ

## thought-garden とは？

thought-garden は、思考を時間をかけて育てるための個人用日記アプリです。

このアプリでは、日々の記録を「完成品」として扱うのではなく、後から見返したり、結びつけたり、編集することで思考そのものを育てていくことを目的としています。

## 最初に作る機能

- [ ] 日記の登録
- [ ] 日記一覧の表示
- [ ] 日記詳細の表示
- [ ] 「今日の教訓」の保存
- [ ] 「今日の教訓」の一覧表示

## このプロジェクトで身に付けたいもの

このプロジェクトは、主に以下の点を練習・検証するためのものです。

- 要件定義・ドキュメント作成
  - 開発メンバーが新規参画したとしてもスムーズに開発できるようなドキュメント作りを心がける
- 開発環境構築
  - 本プロジェクトでは Docker を用いてローカル開発環境を構築している。
  - 業務で一般的な構成 (nginx + php-fpm) を採用し、
  - インフラ構築も含めて理解することを目的としている。
- Domain層の導入
- テスト駆動開発
- コーディング・アーキテクチャに関して、自分なりの正解を見つける
  - 自分の実装に自信を持てるようになるため
